# silver-succotash
readme : # Job Application Tracker

A frontend-only job application tracking dashboard built using *HTML, CSS, and vanilla JavaScript*.

## Features
- Add and manage job applications
- Search, filter, and status-based workflow
- Duplicate application detection
- Inactivity alerts for pending applications
- Client-side analytics without backend
- Persistent storage using browser localStorage

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Browser APIs

## Why This Project
This project demonstrates frontend state management, data modeling, filtering logic, and analytics without relying on backend services.

## How to Run
Simply open index.html in a browser.
